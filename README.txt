SMTP client

Sends a message to a give address or addresses.
All contents should be placed in "message" folder.
In the document "configurations.txt" type all recievers in the first line, separating them by spaces,
type the subject in the second line,
type names of all attachments in the trird, separating them by spaces,
Add all attachments to the "message" folder,
And, finally, type the text of your message in the "text.txt" document.
